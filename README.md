# ProductFinderFromML

It allows you to search and view product details, consuming the MercadoLibre API (developers.mercadolibre.com/items-and-searches)

This is a clean architecture app example built with

- Flow
- LiveData
- Coroutines
- MVVM
- Extension Functions
- Retrofit
- Navigation Components
- DataBinding
